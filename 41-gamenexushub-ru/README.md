# GameNexusHub (RU)

Русскоязычная версия лендинга GameNexusHub, адаптированная для аудитории России.

## Contents
- `index.html` — main page featuring hero, pulse metrics, latest games, quick reviews, immersion spotlights, hardware guides, optimization playbooks, events & streams, stream lineup, community hub, newsletter, FAQ, and footer.
- `style.css` — global styles with neon-inspired dark theme.
- `script.js` — intersection-based progress animations and dynamic year update.
- `_privacy.html` и `_terms.html` — юридические документы на русском.

## Usage
Откройте `index.html` в современном браузере. Дополнительные зависимости не требуются.
