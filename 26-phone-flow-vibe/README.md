# PhoneFlowVibe Landing

Information-first media landing page for PhoneFlowVibe — a digital magazine covering smartphones, mobile applications, and technology trends. Part of the WPgen collection of static marketing experiences.

## Structure
- `index.html` — primary landing page with seven editorial sections.
- `style.css` — clean white and grey layout with blue and orange highlights.
- `script.js` — mobile navigation toggle and hero hover animation.
- `_privacy.html` — placeholder privacy policy page.
- `_terms.html` — placeholder terms of use.

## Local preview
Open `index.html` in your browser or serve the directory with any static file server.
