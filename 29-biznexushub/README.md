# BizNexusHub Landing

Landing platform for BizNexusHub — a US-focused hub for business owners, startup founders, and entrepreneurs seeking analytics,
mentorship, and partner support.

## Structure
- `index.html` — marketing landing page with twelve themed sections covering the platform, services, metrics, and community.
- `style.css` — light corporate UI with bold blue and orange accents, designed on a clean grid.
- `script.js` — mobile navigation toggle and interactive hero parallax effect.
- `_privacy.html` — placeholder privacy policy page.
- `_terms.html` — placeholder terms of service.

## Local preview
Open `index.html` in your browser or serve the directory with any static file server.
