# SoftFlowVibe Landing Page

Lightweight marketing site for SoftFlowVibe, an IT company building UX-first software and SaaS products.

## Structure
- `index.html` — main landing page
- `style.css` — global styles
- `script.js` — navigation interactions and parallax helper
- `_privacy.html`, `_terms.html` — legal pages

## Development
Open `index.html` in a modern browser. No build step is required.
