# ConsultFlowVibe Landing

Calming, flow-inspired landing page for the coaching and consulting platform ConsultFlowVibe. Part of the WPgen collection of modern marketing microsites.

## Structure
- `index.html` — single-page layout featuring hero, about, programs, methods, flow roadmap, corporate solutions, resources, benefits, success stories, and contact sections.
- `style.css` — airy pastel styling with mint and pale blue gradients, glassmorphism accents, and responsive grids.
- `script.js` — lightweight navigation toggle for mobile viewports.
- `_privacy.html` — placeholder privacy policy document.
- `_terms.html` — placeholder terms & conditions.

## Local preview
Open `index.html` in any modern browser or serve the folder via a static file server to preview the experience.
