# ConsultNexusHub Landing

Corporate tech-inspired landing page for ConsultNexusHub, a global marketplace of consulting talent connecting experts, startups, and enterprises.

## Structure
- `index.html` — single-page layout with hero, about, services marketplace, how-it-works, success stories, why choose us, platform stats, community, talent spotlight, resources, insights, and contact sections.
- `style.css` — business-minimalist styling in white, gray-blue, and lime accent tones with tech-forward patterns.
- `script.js` — mobile navigation toggle logic.
- `_privacy.html` — placeholder privacy policy content.
- `_terms.html` — placeholder terms and conditions.

## Local preview
Open `index.html` in a modern browser or serve the folder using any static file server to explore the experience.
