# SmartStrategyAdvisorGo Landing

Premium landing page for the strategic consulting firm SmartStrategyAdvisorGo. Built as part of the WPgen collection of static marketing sites.

## Structure
- `index.html` — main landing page with seven content sections.
- `style.css` — premium minimal styling in dark blue, white, and silver tones.
- `script.js` — navigation toggle and hero parallax interactions.
- `_privacy.html` — privacy policy placeholder.
- `_terms.html` — terms & conditions placeholder.

## Local preview
Open `index.html` directly in a browser or serve the directory via any static file server.
