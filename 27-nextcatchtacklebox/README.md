# NextCatchTackleBox Landing Page

Landing page for **NextCatchTackleBox** (US market, English) highlighting premium fishing tackle kits, curated gear, and educational resources.

## Structure
- `index.html` — Main single-page site with 11 sections, including hero, about, kits, catalog, guides, reviews, guarantee, and contact.
- `style.css` — Custom styling inspired by natural tones (olive, teal, sand, wood).
- `script.js` — Navigation toggle, subtle hero parallax, and lightweight form handlers.
- `_privacy.html`, `_terms.html` — Placeholder legal documents linked in the footer.
- `images/` — Directory reserved for future local assets.

## Getting Started
Open `index.html` in your browser to view the landing page. No build steps are required.

## Customization Tips
- Update image URLs with branded photography or locally hosted assets.
- Adjust accent colors in `:root` variables within `style.css` to match brand guidelines.
- Replace placeholder links and form actions with production endpoints as needed.
