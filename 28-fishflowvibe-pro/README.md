# FishFlowVibe Landing

Inspiring media landing page for FishFlowVibe, a U.S.-based digital magazine celebrating fishing culture, aquatic ecology, and the mindful "flow" lifestyle.

## Structure
- `index.html` — single-page experience with hero, highlights, story, philosophy, articles, seasonal calendar, guides, eco awareness, gear reviews, community, newsletter, and footer sections.
- `style.css` — mobile-first styling with soft gradients, organic lines, and responsive grids inspired by lakeside palettes.
- `script.js` — lightweight JavaScript for navigation toggling on small screens and smooth in-page navigation.
- `_privacy.html` — placeholder privacy policy page.
- `_terms.html` — placeholder terms & conditions page.

## Local preview
Open `index.html` in a browser or serve the folder with any static file server to explore the landing page.
