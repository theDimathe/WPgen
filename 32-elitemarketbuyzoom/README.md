# EliteMarketBuyZoom Landing

Luxury e-commerce landing page for EliteMarketBuyZoom, a boutique showcase of premium technology and accessories. Part of the WPgen collection of static marketing experiences.

## Structure
- `index.html` — primary landing with thirteen luxury-focused sections.
- `style.css` — dark, gilded, minimal aesthetic styling.
- `script.js` — mobile navigation toggle and scroll interactions.
- `_privacy.html` — privacy policy placeholder.
- `_terms.html` — terms & conditions placeholder.

## Local preview
Open `index.html` in a browser or serve the directory with any static file server.
