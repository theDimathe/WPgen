# EliteSmartphoneZoom Landing

Premium landing page for the EliteSmartphoneZoom flagship smartphone boutique. Part of the WPgen collection of static marketing experiences.

## Structure
- `index.html` — main landing page with seven luxury-themed sections.
- `style.css` — monolithic dark styling with metallic gradients and luminous accents.
- `script.js` — navigation toggle, smooth scrolling, and hero zoom interaction.
- `_privacy.html` — privacy policy placeholder.
- `_terms.html` — terms & conditions placeholder.

## Local preview
Open `index.html` directly in a browser or serve the directory with any static file server.
