# ShopNexusHub

Landing page número 37 localizada para Chile y traducida al español.

## Contenido
- `index.html` — página principal con secciones de hero, quiénes somos, funcionamiento, categorías, tiendas, foco local, beneficios, ofertas, testimonios, recursos, formulario de socios y contacto.
- `style.css` — estilos globales del sitio.
- `script.js` — desplazamiento suave y lógica de modales.
- `_privacy.html` y `_terms.html` — documentos legales en español.

## Uso
Abre `index.html` en un navegador moderno para visualizar el sitio. No requiere dependencias adicionales.
